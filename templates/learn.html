Ports are virtual points where network connections start and end. Each port is tied to a specific process or service. They're crucial in cybersecurity and network communication.

Common Ports

  • Port 80: HTTP (Web traffic)
  • Port 443: HTTPS (Secure web)
  • Port 22: SSH (Remote access)
  • Port 53: DNS (Domain Name System)
  • Port 25: SMTP (Email sending)
