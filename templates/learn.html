PSX Learn Mode

Understand common ports and their potential risks

Port 21 – FTP

Used for file transfers, but transmits data unencrypted. Vulnerable to sniffing and brute force attacks. Consider replacing with SFTP.

Port 22 – SSH

Secure shell for remote access. Keep updated and use strong authentication to avoid brute force risks.

Port 80 – HTTP

Unsecured web traffic. Should redirect to HTTPS (port 443) to prevent MITM attacks.

Port 443 – HTTPS

Secure web traffic. Ensure certificates are valid and modern protocols (TLS 1.2+) are used.

Port 3389 – RDP

Remote Desktop Protocol. Often targeted. Should be firewalled and monitored closely.
